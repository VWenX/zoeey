Web开发辅助工具,提供大量Php常用函数替代类。

本框架最体现亲Php开发者的包。 {@link org.zoeey.util.Base64 Base64} 实现了base64_encode()、base64_decode()。 {@link org.zoeey.util.BinaryFileHelper BinaryFileHelper}， {@link org.zoeey.util.TextFileHelper TextFileHelper} 实现了file_get_contents()、file_put_contents()，且可以方便的对数据进行追加操作。 {@link org.zoeey.util.DirInfo DirInfo} 可方便的获取网站目录下各资源的绝对路径。 {@link org.zoeey.util.EncryptHelper EncryptHelper}对应md5()、md5_file()、sha1()、sha1_file()。 {@link org.zoeey.util.EnvInfo EnvInfo} 可以获取运行环境变量。 {@link org.zoeey.util.EnvInfo FileHelper} 提供文件、文件夹的创建删除和移动等操作。 {@link org.zoeey.util.EnvInfo ImageHelper} 制作缩略图，为图片打水印都可以使用此类。 {@link org.zoeey.util.EnvInfo IpHelper} IP转换工具。建议使用long型存储IP，节省空间便于检索。 {@link org.zoeey.util.EnvInfo JsonHelper} 其由JsonEncoder与JsonDecoder组成，类似json_encode().json_decode()。 在强类型语言中json decode操作稳定性较于Php而言要差很多，所以在解码操作中请尽量关注验证信息{@link org.zoeey.util.JsonDecoder.Validator JsonDecoder.Validator}。 {@link org.zoeey.util.KeyValueFile KeyValueFile} 提供了一个结构较为松散的键值对文件结构分析工具，可用于简单的配置文件。 {@link org.zoeey.util.LangHelper LangHelper} 语言包调用，使用了KeyValueFile文件格式。 {@link org.zoeey.util.NumberHelper NumberHelper} 数字进制转换。 {@link org.zoeey.util.NumberHelper ParamHelper} 多数情况下可以合并多个布尔型参数，节省编码。 {@link org.zoeey.util.NumberHelper QueryStringHelper} 请求字符串分析,类似于 parse_str()。 {@link org.zoeey.util.NumberHelper RandomHelper} 随机数，随机字符串生成。 {@link org.zoeey.util.NumberHelper ReflectCacheHelper} 反射缓存辅助，方便获取类信息。 {@link org.zoeey.util.Snoopy Snoopy} 提供HTTP模拟访问。 {@link org.zoeey.util.StringHelper StringHelper} 字符串常规操作辅助。 {@link org.zoeey.util.TimeFormatHelper TimeFormatHelper} 时间格式化辅助。 {@link org.zoeey.util.UrlBuilder UrlBuilder} 如http_build_url()，生成请求字符串。 {@link org.zoeey.util.UrlHelper UrlHelper} URL encode和decode操作。
